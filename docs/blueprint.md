# **App Name**: Cattle Health 360

## Core Features:

- Login Page: Secure entry point with dummy Farm ID and Password fields, redirecting to the main dashboard.
- Dashboard Layout: Glassmorphism design with dark slate backgrounds, translucent cards, and neon emerald/blue accents, featuring 3D hover effects and responsive design.
- Herd Snapshot: Summary cards displaying Total Head, Critical Alerts, In Heat, and Overall Health Score.
- Health Alerts Feed: Live feed of alerts (Fever, Geofence, etc.) to monitor cattle health in real-time.
- Vitality KPIs: Display of key performance indicators such as Average Temperature, Rumination, and Milk Yield.
- Sensor Health Check: Detailed list of active sensors with status (Online/Offline) and Calibration status.
- AI Health Assistant: An AI assistant that is a tool to respond to common alert events and provides static advice.

## Style Guidelines:

- Primary color: Neon emerald (#39FF14) for a vibrant, high-tech feel.
- Background color: Dark slate gray (#2E2E2E) to enhance the glassmorphism effect.
- Accent color: Neon blue (#14D4FF) to highlight interactive elements and data visualizations.
- Body and headline font: 'Inter', a sans-serif font, for a modern and neutral look that ensures readability across the dashboard.
- Utilize 'lucide-react' icons for clear and consistent visual representation.
- Implement a responsive layout that fills the screen, ensuring optimal viewing on various devices.
- Use Framer Motion for smooth transitions and 3D hover effects on cards, enhancing user engagement.
